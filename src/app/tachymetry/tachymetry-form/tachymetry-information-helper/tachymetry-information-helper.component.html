How to create tachymetry stations file?
Only supported file type is txt.
Each station is separated by empty line.
Single station must contain four lines which describe main data of station.
First line is a station number. It supports only integers. This number has to be unique.
Second line is a station name
Third line is a starting geodetic network point. The supported format of this point is the same like in default coordinates file format.
First column has name of point. Second column is a coordinate X given in 2000 and third column is a coordinate Y in the same coordinates system.
Forth line is a ending geodetic network point. The format of this line is the same as above.
After this main station's information file should contain lines regarding to pickets measured data.
Before those lines there should be an empty line placed between.
Format of pickets measured data: each line has three columns.
1. Picket name
2. Coordinate X in 2000
3. Coordinate Y in 2000
