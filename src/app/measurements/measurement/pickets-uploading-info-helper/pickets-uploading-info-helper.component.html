How to create pickets file to upload?
Only supported file type is txt.
The first column is supposed to have picket name,
the second - value of coordinate X in geodetic coordinates system 2000 and
the third one - value of coordinate Y in the same system.
Each picket should be placed in separate line and
each column should have one space break between each other.

Example:

picket1 5575321.99 6563290.64
picket2 5575381.99 6563220.64
